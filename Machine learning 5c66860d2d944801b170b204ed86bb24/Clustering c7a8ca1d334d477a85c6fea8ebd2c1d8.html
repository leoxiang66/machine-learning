Intro

Unsupervised learning

Given some unlabeled data {xi}\left\{\boldsymbol{x}_{i}\right\}{xi​}﻿ we want to discover latent structure in it.

Clustering

group objects xix_{i}xi​﻿ into KKK﻿ groups (clusters) based on their similarity

Problem definition

  • Given: collection of data points X={x1,…,xN}⊂X\boldsymbol{X}=\left\{\boldsymbol{x}_{1}, \ldots, \boldsymbol{x}_{N}\right\} \subset \mathbb{X}X={x1​,…,xN​}⊂X﻿
  • Goal: For each object xix_{i}xi​﻿ find the corresponding assignment zi∈{1,…,K}z_{i} \in\{1, \ldots, K\}zi​∈{1,…,K}﻿ to one of the KKK﻿ groups (clusters). such that:
    • objects within the same groupd are similar to each other
    • objects between different groups are dissimilar from each other

Model

  • Each of KKK﻿ clusters is defined by its centroid μk∈RD\boldsymbol{\mu}_{k} \in \mathbb{R}^{D}μk​∈RD﻿
  • cluster indicator zi∈{0,1}K\boldsymbol{z}_{i} \in\{0,1\}^{K}zi​∈{0,1}K﻿ wityh zi=1⟺xi\boldsymbol{z}_{i}=1 \Longleftrightarrow \boldsymbol{x}_{i}zi​=1⟺xi​﻿ belongs to cluster iii﻿ (i.e. one-hot encoding)

Minimizing the K-means objective

  • The joint optimization problem is difficult to solve (discrete optim.; even if ZZZ﻿ is assumed to continuous it is non-convex)

  • However, the two subproblems
    • min⁡ZJ(X,Z,μ)\min _{Z} J(\boldsymbol{X}, \boldsymbol{Z}, \boldsymbol{\mu})minZ​J(X,Z,μ)﻿
    • min⁡μJ(X,Z,μ)\min _{\boldsymbol{\mu}} J(\boldsymbol{X}, \boldsymbol{Z}, \boldsymbol{\mu})minμ​J(X,Z,μ)﻿

    have simple closed form solutions!

  • idea: alternating optimization → update ZZZ﻿ and μ\muμ﻿ in turns!
    • this method for solving KKK﻿-means is called LIoyd's algorithm

LIoyd's algorithm for K-means

  1. initialize the centroids μ={μ1,…,μK}\boldsymbol{\mu}=\left\{\boldsymbol{\mu}_{1}, \ldots, \boldsymbol{\mu}_{K}\right\}μ={μ1​,…,μK​}﻿
  1. update cluster indicators (solve min⁡ZJ(X,Z,μ)\min _{\boldsymbol{Z}} J(\boldsymbol{X}, \boldsymbol{Z}, \boldsymbol{\mu})minZ​J(X,Z,μ)﻿)
    • 对每一个sample, 找到质心距离他最近的cluster, 并将对应的indicator设为1
  1. update centroids (solve min⁡μJ(X,Z,μ)\min _{\boldsymbol{\mu}} J(\boldsymbol{X}, \boldsymbol{Z}, \boldsymbol{\mu})minμ​J(X,Z,μ)﻿)
    • 在把每个sample分组以后， 更新分组cluster的质心
  1. if objective J(X,Z,μ)J(\boldsymbol{X}, \boldsymbol{Z}, \boldsymbol{\mu})J(X,Z,μ)﻿ has not converged, return to step 2

Limitations of K-means

It is important to distinguish between the model (distortion J(X,Z,μ)J(\boldsymbol{X}, \boldsymbol{Z}, \boldsymbol{\mu})J(X,Z,μ)﻿ in our case) and the algorithm used to fit it (here Lloyd's algorithm).

  • Modeling issues
    • Underlying assumptions are not explicit
    • Can't detect clusters with overlapping convex hulls
    • Sensitivity to outliers
    • No uncertainty measure

  • Algorithmic issues
    • Extreme sensitivity to initialization

⇒ switch to more principled probabilistic framework!

Gaussian mixture model (GMM)

We decompose the joint probability p(x,z∣θ)\mathrm{p}(\boldsymbol{x}, \boldsymbol{z} \mid \boldsymbol{\theta})p(x,z∣θ)﻿ using the product rule

In Gaussian mixture model we assume:

  • cluster prior is categorical
  • each cluster has its own multivariate normal distribution
  • the parameters consist of the set of parameters of the prior and all conditional distributions θ={π,μ,Σ}\theta=\{\pi, \mu, \Sigma\}θ={π,μ,Σ}﻿
